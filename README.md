## Installation
### pip
```shell
pip install tiny-algos
```
### Poetry
- install from source
    ```shell
    poetry install
    ```
- add to your environment
    ```shell
    poetry add tiny-algos
    ```
## Supported Data Structures / Algorithms
### Collections
- multiset.py
    - MultiSet
- minmultiset.py
    - MinMultiSet
    - MaxMultiSet
### Arithmetics
- mod_arithmetic.py
    - mod_factorial
    - mod_permutation
    - mod_combination
